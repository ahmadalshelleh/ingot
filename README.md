to run the project:

    After clone the project
    open terminal
        1) composer install
        2) npm install
        3) npm run dev

    Open second terminal
        1) modify env file
        2) php artisan key:generate
        2) php artisan migrate
        3) php artisan db:seed --class=AdminSeeder
        4) php artisan serve
